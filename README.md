# MGM Residency - Hotel & Resort Website

A React.js hotel landing page inspired by the Hotelux design, built for **MGM Residency, Chikmagaluru**.

## Tech Stack
- React 18
- CSS (custom variables, no framework)
- Google Fonts: Cormorant Garamond + Jost

## Sections
1. **Navbar** — Fixed, scrolled effect, dropdowns, cart badge
2. **Hero** — Full-screen with background image, animated headline, play button
3. **About** — Two-column layout with image gallery and features
4. **Facilities** — 4-card grid: Taxi, Pool, Tourist Guide, Bike Rental
5. **Rooms & Suites** — 3 room cards with price badges and amenity icons
6. **Testimonials** — Paginated 2-column review cards with avatar
7. **Footer** — 4-column layout with social icons, quick links, contact

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000)

## Color Palette
- Primary Dark: `#12111e`
- Gold Accent: `#c9a97a`
- Font Serif: Cormorant Garamond
- Font Sans: Jost

## Customization
- Replace Unsplash image URLs in each component with your actual hotel photos
- Update hotel name, pricing, and contact details in the respective component files
- Adjust colors in `src/index.css` under `:root`
